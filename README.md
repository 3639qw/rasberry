# 📊 AI Security Camera

> **YOLOv5** 기반의 사람 감지 프로그램
---

## 📝 프로젝트 개요
- AI Security Camera는 **라즈베리파이**와 **YOLOv5** 모델을 활용하여 사람을 자동으로 감지하고, 이를 사용자에게 알림으로 전달하는 시스템입니다.  
- 감지된 사람에 대해 **비디오 촬영**과 **사진 촬영** 기능을 제공하며, 보안 감시가 필요한 환경에서 실시간 감지 및 기록을 지원합니다.

---

## 🛠 기술 스택
| 구분           | 사용 기술                                       |
|----------------|---------------------------------------------|
| **Hardware**   | Raspberry Pi 4, Raspberry Pi Camera Module |
| **AI Model**   | YOLOv5 (PyTorch 기반)                       |
| **Frontend**   | Python (OpenCV, Tkinter, Pygame 등)          |
| **Backend**    | Python 3.x                                   |
| **알림**       | MQTT 알림         |
| **버전 관리**  | GitHub                                        |

---

## 📦 주요 기능
- **사람 감지**
    - YOLOv5 모델을 활용하여 실시간으로 사람을 감지하고, 감지된 사람을 화면에 표시
    - 감지된 사람의 위치와 시간 정보 로그 기록
- **비디오 촬영**
    - 사람 감지 후 자동으로 비디오 촬영 시작
    - 일정 시간 후 비디오 종료 및 저장 (영상 파일 포맷: MP4)
- **사진 촬영**
    - 감지된 사람을 기반으로 자동으로 사진 촬영
    - 촬영된 이미지는 자동으로 저장 (파일 포맷: JPEG)
- **알림 시스템**
    - 사람 감지 시 MQTT 메시지 수신
- **보안**
    - 촬영된 파일 및 로그는 로컬 디스크에 저장되며, 파일 암호화 및 접근 제어 기능 제공
    - 환경 변수 파일로 중요한 설정 정보 보호
